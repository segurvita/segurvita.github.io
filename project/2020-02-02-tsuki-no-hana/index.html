Tsuki no Hana

Feb 2, 2020
Tsuki no Hana

Global Game Jam 2020に参加した際、4人のチームで開発したゲームです。

ルービックキューブのような立体的なパズルは、2Dのディスプレイでは表現が難しいですが、Looking Glassならそのまま立体視で表現できるのでは？という考えから、立体パズルゲームを作りました。

完成したのがこちらです。

今回のグローバルゲームジャムで作ったゲームです！テ○リスとウボ○ゴ3Dが融合したようなルール！通信対戦もできます！すごく楽しかった！おつかれさまでした！#ggjmd #ggj20 #ggj20jp #ルキグラス　#LookingGlass pic.twitter.com/DchDKFPNqx

— segur(せぎゅ)@STYLYエンジニア (@segur_vita) February 2, 2020

パズルゲームを開発した経験がなかったので、ブロックの設置や衝突の判定に苦労しましたが、何とかメンバーの助けを借りて実装できました。

概要

  • 開発環境: Unity
  • 動作環境: Looking Glass
  • 担当箇所: ハードウェア準備、一部ソフトウェア開発（パズル落下制御等）
